# RedRoom_page_CTF

Statyczna strona startowa do fikcyjnego scenariusza CTF „Czerwona Rzeka”. Nie wymaga serwera ani instalacji: na GitHub Pages jako źródło wybierz katalog główny repozytorium.

## Podmiana obrazu

1. Umieść swój finalny plik JPG (z przygotowaną przez Ciebie zawartością CTF) w `assets/`.
2. W `index.html` zmień wartość `src` znacznika `<img>` na nazwę tego pliku.
3. Opcjonalnie zaktualizuj opis `alt`, aby odpowiadał nowemu obrazowi.

Aktualny `assets/room-signal-placeholder.png` jest wyłącznie bezpiecznym placeholderem. Dwa wskazane tropy są celowo obecne tylko w zakodowanej postaci w kodzie źródłowym strony.
